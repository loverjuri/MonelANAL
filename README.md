# Модуль учёта ЗП и финансов

Персонализированная система для автоматического расчёта накопленной заработной платы, учёта рабочего времени (основная и вторая работа), контроля расходов и прогнозирования бюджета.

**Платформа:** Google Sheets + Google Apps Script (GAS) + Telegram Bot API  
**Часовой пояс:** Europe/Moscow (UTC+3)

## Структура проекта

- **src/** — исходный код Google Apps Script (копировать в редактор GAS или деплой через clasp)
- **docs/SETUP.md** — пошаговая инструкция по настройке таблицы, бота и webhook

## Быстрый старт

1. Создайте Google Таблицу и настройте листы по схеме из [docs/SETUP.md](docs/SETUP.md).
2. Создайте проект Google Apps Script, привязанный к этой таблице.
3. Скопируйте содержимое файлов из `src/` в соответствующие файлы проекта GAS (или используйте clasp).
4. Запустите один раз функции `initSheetIfNeeded()` и `initConfigDefaultsIfEmpty()` для создания листов и строк Config по умолчанию (если не создали вручную).
5. Укажите Bot Token и Chat ID в свойствах скрипта (Файл → Свойства проекта → Свойства скрипта) или в листе Config.
6. Разверните как веб-приложение (doGet/doPost) и настройте webhook Telegram на URL развёрнутого скрипта.
7. Вызовите `installTriggers()` один раз для установки триггеров по расписанию.

Подробности — в [docs/SETUP.md](docs/SETUP.md).
